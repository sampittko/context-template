# Artifacts

`3-artifacts/` stores outputs produced by this context.

Suggested buckets:

- `3-artifacts/docs/`: generated or authored deliverable docs
- `3-artifacts/projects/`: project briefs, specs, or output packages
- `3-artifacts/prompts/`: reusable prompts and workflows
- `3-artifacts/skills/`: context-produced skills or skill packs

Rule:

- `3-artifacts/` is not source evidence by default.
- If an artifact later becomes canonical input, promote it explicitly into `1-raw/`.
